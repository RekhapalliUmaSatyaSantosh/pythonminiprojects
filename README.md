# Python Mini Projects

A collection of small Python projects covering games, utilities, and machine learning basics — built for practicing core Python concepts, OOP, and intro-level ML.

## 🎮 Games
| Project | Description |
|---|---|
| [`Adventure_game.py`](./Adventure_game.py) | Text-based choose-your-own-adventure game |
| [`Hangman_game.py`](./Hangman_game.py) | Classic word-guessing game with limited attempts |
| [`Quiz_game.py`](./Quiz_game.py) | Multiple-choice quiz with score tracking |
| [`math_quizz.py`](./math_quizz.py) | Randomized arithmetic quiz generator |
| [`Dice_game.py`](./Dice_game.py) | Dice-based game with win/loss logic |
| [`Dice_Rolling.py`](./Dice_Rolling.py) | Simple dice roll simulator |
| [`Rock_Paper_Scissors.py`](./Rock_Paper_Scissors.py) | Classic Rock-Paper-Scissors vs. computer |
| [`Number_Guessing.py`](./Number_Guessing.py) | Number guessing game with hints |
| [`slot_machine.py`](./slot_machine.py) | Slot machine simulator with payout logic |
| [`chess.py`](./chess.py) | Basic chess board/move logic |

## Getting Started

```bash
git clone https://github.com/RekhapalliUmaSatyaSantosh/pythonminiprojects.git
cd pythonminiprojects
python <project_name>.py
```

Some scripts (e.g. `QRCode_Generator.py`, ML scripts) require extra packages:

```bash
pip install qrcode scikit-learn pandas numpy
```

## Tech Stack
- **Language:** Python 3
- **Libraries used across projects:** `scikit-learn`, `qrcode`, `random`, `datetime`
