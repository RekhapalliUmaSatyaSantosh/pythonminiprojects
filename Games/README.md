# 🎮 Games

A collection of small terminal-based games built in Python. Great for practicing loops, conditionals, randomization, and basic game logic.

## Projects

| File | Description |
|---|---|
| `Adventure_game.py` | A text-based forest adventure game with story choices. |
| `Dice_Rolling.py` | Simulates rolling a dice and tracks the number of rolls. |
| `Dice_game.py` | Simple dice-roll function/game logic. |
| `Hangman_game.py` | Classic Hangman — guess the secret word letter by letter before running out of attempts. |
| `Number_Guessing.py` | Guess a randomly generated number between 1 and 100. |
| `Quiz_game.py` | An interactive quiz game with a yes/no start prompt. |
| `Rock_Paper_Scissors.py` | Rock-Paper-Scissors against the computer, with emoji output and score tracking. |
| `Suduko_Validation.py` | Validates whether a given Sudoku board follows the rules (rows, columns, boxes). |
| `chess.py` | Reference/demo of chess piece movement rules (e.g. King). |
| `mastermind_game.py` | A Mastermind-style number-guessing game with a secret code. |
| `math_quizz.py` | Randomly generated math problems (+, -, *) to test quick arithmetic. |
| `slot_machine.py` | A slot machine simulator with betting, lines, and payouts. |

## How to Run
Each file is standalone. Run any game with:
```bash
python <filename>.py
```

## Requirements
Most games use only Python's standard library (`random`, `time`). No extra installation needed.
