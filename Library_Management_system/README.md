# 📚 Library Management System

A console-based **Library Management System** developed in **Python** using **SQLite**. The application helps manage books, members, and book transactions through a simple command-line interface.

---

## 🚀 Features

### 📖 Book Management
- Add new books
- View all books
- Search books by ID or title
- Update book details
- Delete books

### 👤 Member Management
- Add new members
- View all members
- Search members
- Update member details
- Delete members

### 🔄 Book Transactions
- Issue books to members
- Return issued books
- View currently issued books
- Search issued book records
- View pending returns

### 📊 Dashboard
Displays:
- Total books
- Total members
- Total books issued
- Total books returned
- Pending returns

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/RekhapalliUmaSatyaSantosh/pythonminiprojects.git
```

Navigate to the project folder:

```bash
cd pythonminiprojects/Library_management
```

Run the application:

```bash
python library.py
```
