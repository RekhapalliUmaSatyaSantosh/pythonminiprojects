# 🛠️ Utilities

Small, practical scripts that solve one focused task each — good everyday tools and beginner-friendly Python practice.

## Projects

| File | Description |
|---|---|
| `Acronyms.py` | Converts a typed phrase into its acronym (first letters of each word, capitalized). |
| `Digitalclock.py` | A live digital clock / time tools display in the terminal. |
| `QRCode_Generator.py` | Generates a QR code image from user-provided text or a URL. |
| `StrongPassword.py` | Checks password strength and gives a score with feedback; can also generate strong passwords. |
| `Unit_converter.py` | Converts between units — length, weight, and temperature. |
| `Word_count.py` | Counts words in a block of user-entered text. |

## Requirements
- `QRCode_Generator.py` needs the `qrcode` package:
```bash
  pip install qrcode[pil]
```
- All other scripts use only the Python standard library — no installation required.

## How to Run
```bash
python <filename>.py
```
Each script runs independently and prompts for input in the terminal.
